## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/vervixsolutions/crypto33-Masters.git
```

2. Open `index.html` in your web browser to view the homepage
3. Navigate to `training.html` for the training page
4. Navigate to `freedom-hub.html` for the news aggregation page

## Pages

- **Homepage** (`index.html`): Main landing page with hero section, vault teaser, and membership information
- **Training** (`training.html`): 8-week training program with interactive lessons
- **Freedom Hub** (`freedom-hub.html`): News aggregation platform with multi-perspective analysis

## Design Philosophy

Crypto33 Masters promotes:
- **Truth**: Multiple perspectives on news and events
- **Education**: Structured learning paths for cryptocurrency mastery
- **Freedom**: Financial sovereignty and wealth building
- **Community**: Respectful discussion and knowledge sharing
